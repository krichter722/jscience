Requirements:

This library uses the Javolution high-performance framework either in the classpath (JDK1.5+) or in the bootclasspath (J2ME, JRE1.1 .. JRE1.4). For example: 

JRE1.5+:  java -cp javolution.jar;jscience.jar ...

Others:   java -Xbootclasspath/a:javolution.jar -cp jscience.jar ...
